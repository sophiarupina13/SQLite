# SQLite
A pet project with a realization of server that has a homepage (GET /), login page (GET and POST /login), signup page (GET and POST /signup) and can shows database with pagination (GET /comments) or the last added row (GET /lastcomment). Also rows in database can be added (POST /comments), changed (PATCH /comments[id]) and deleted (DELETE /comments[id]) only if there is a current session on the server (login successful).
